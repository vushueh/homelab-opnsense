# Project 03 Troubleshooting

Use this folder for Suricata IDS/IPS break/fix notes.

## Break/Fix Ideas

- IDS enabled on the wrong interface: no alerts appear.
- Rule source enabled but rules not downloaded.
- IPS blocking mode enabled too early and blocks valid lab traffic.
- Alerts appear but are not forwarded to the SOC stack.

## Log Format

```text
## Issue — YYYY-MM-DD
### Symptom
### Root Cause
### Fix
### Verification
### Lesson Learned
```
